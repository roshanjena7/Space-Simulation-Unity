# 🚀 Space Simulation (Unity)

A real-time space simulation built using Unity.

## ✨ Features
- 🚀 Rocket launch → orbit transition
- 🛰 Orbit stability detection (Stable / Drifting / Unstable)
- 📡 Signal strength system with UI
- 📊 Mission Control Dashboard
- 🎥 Cinematic camera follow

## 🛠 Tech Stack
- Unity 6
- C#
- TextMeshPro
